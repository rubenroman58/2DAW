Ejercicio 4

Crea una página web con Bootstrap que contenga:
  1. EjUn navbar con una cabecera y al menos tres enlaces
  2. Un carrusel en la cabecera con al menos cuatro imágenes.
  3. Un grid de 6 contenedores con la siguiente distribución responsive :
    1. Escritorio (lg en adelante): 6 columnas por fila (6)
    2. Tableta (md): 3 columnas por fila (3-3)
    3. Móvil (sm y xs): 2 columnas por fila (2-2-2)
  4. Dentro de cada contenedor, Debes agregar un elemento card con su secciones
    1. card-body
    2. card-title
    3. card-text
    y un boton predeterminado de bootstrap (primary, info ... etc)
  5. Configurar las medias query del footer para que cambie de color cuando la pantalla tenga menos de 1000px
