Ejercicio 15

Método Devuelve Descripción
RegExp.exec(String) Array o null Devuelve un array con la coincidencia o null si no se encuentra.
RegExp.test(String) Booleano (true o false) Devuelve true si hay coincidencia, false si no la hay.
String.match(RegExp) Array o null Devuelve un array con todas las coincidencias o null si no hay coincidencias.
String.matchAll(RegExp) Iterador Devuelve un iterador con todas las coincidencias y grupos de captura.
String.search(RegExp) Índice (número) Devuelve el índice de la primera coincidencia o -1 si no la encuentra.
String.replace(RegExp) Cadena Devuelve una nueva cadena con las coincidencias reemplazadas por una nueva subcadena.
String.replaceAll(RegExp) Cadena Devuelve una nueva cadena reemplazando todas las coincidencias.
String.split(RegExp) Array Devuelve un array dividiendo la cadena en partes según el patrón de la expresión regular.

Bandera Descripción
d Permite hacer una búsqueda en el texto de forma que se respeten los saltos de línea, buscando más allá de las fronteras de la línea.
g Realiza una búsqueda global, es decir, encuentra todas las coincidencias dentro del texto, no solo la primera.
i Hace que la búsqueda no sea sensible a mayúsculas o minúsculas (es decir, que "a" y "A" se consideren iguales).
