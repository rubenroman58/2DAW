Sintaxis Nombre Uso Significado Ejemplo
= Asignación Asigna el valor de la derecha a la variable de la izquierda a = 10 asigna el valor 10 a la variable `a` var a = 10;
+= Asignación con suma Suma el valor de la derecha al valor de la izquierda y asigna el resultado a += 5 es equivalente a a = a + 5 var a = 10; a += 5; // a = 15
-= Asignación con resta Resta el valor de la derecha al valor de la izquierda y asigna el resultado a -= 3 es equivalente a a = a - 3 var a = 10; a -= 3; // a = 7
*= Asignación con multiplicación Multiplica el valor de la derecha por el de la izquierda y asigna el resultado a *= 2 es equivalente a a = a * 2 var a = 10; a *= 2; // a = 20
/= Asignación con división Divide el valor de la izquierda por el valor de la derecha y asigna el resultado a /= 2 es equivalente a a = a / 2 var a = 10; a /= 2; // a = 5
%= Asignación con módulo Divide el valor de la izquierda por el de la derecha y asigna el resto a %= 3 es equivalente a a = a % 3 var a = 10; a %= 3; // a = 1
<<= Asignación con desplazamiento a la izquierda Desplaza los bits de la izquierda a la izquierda el número de posiciones indicado a <<= 1 desplaza los bits de `a` una posición a la izquierda var a = 10; a <<= 1; // a=20
>= Mayor o igual que Compara si el valor de la izquierda es mayor o igual que el valor de la derecha a >= 5 devuelve `true` si `a` es mayor o igual a 5 var a = 10; a >= 5; // true
>>= Asignación con desplazamiento a la derecha Desplaza los bits de la izquierda a la derecha el número de posiciones indicado a >>= 1 desplaza los bits de `a` una posición a la derecha var a = 10; a >>= 1; // a = 5
>>>= Asignación con desplazamiento lógico a la derecha Desplaza los bits a la derecha rellenando con ceros, independientemente del signo a >>>= 1 desplaza los bits de `a` una posición a la derecha rellenando con ceros var a = -10; a >>>= 1; // resultado positivo
&= Asignación con AND Realiza un AND bit a bit entre los dos operandos y asigna el resultado a &= b realiza la operación AND bit a bit entre `a` y `b` var a = 15; a &= 9; // a = 9
|= Asignación con OR Realiza un OR bit a bit entre los dos operandos y asigna el resultado a |= b realiza la operación OR bit a bit entre `a` y `b` var a = 15; a |= 9; // a = 15
^= Asignación con XOR Realiza un XOR bit a bit entre los dos operandos y asigna el resultado a ^= b realiza la operación XOR bit a bit entre `a` y `b` var a = 15; a ^= 9; // a = 6
[] Asignación de acceso a arrays Accede o asigna valores en un array usando su índice a[0] accede al primer elemento de `a` var a = [1, 2, 3]; a[0] = 10; // a = [10, 2, 3]
